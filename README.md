# HR Dashboard (Power BI)

An interactive and visually rich **HR Analytics Dashboard** built using **Power BI**.  
This dashboard helps HR teams analyze workforce composition, salary distribution, hiring and termination trends, and department/position insights.

---

## Project Overview

This HR Dashboard provides a complete view of employee-related metrics using clean and well-structured visuals.  
It is built from the dataset `HRDataset_v14.xlsx` and packaged as a reusable Power BI template (`HR Dashboard.pbit`).

### Key Features
- **Employee Overview KPIs**
  - Total Employees
  - Total Salary Distribution
- **Department Analysis**
  - Employee Count by Department (Donut Chart)
- **Gender-Based Salary Analysis**
  - Salary Distribution for Male vs Female Employees
- **Position-Level Insights**
  - Employee Count by Position
- **Hiring & Termination Trends**
  - Year-wise line chart for hire/termination patterns
- **Geographical Distribution**
  - Map visual of employee states across the US
